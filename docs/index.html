WhisperApp

macos · menu-bar · dictation · whisper + llm

พูดแล้วให้ Mac พิมพ์ให้ — กด Fn ค้างแล้วพูด ปล่อยปุ๊บ ข้อความเด้งลงแอปที่ใช้อยู่ทันที

⬇︎  ดาวน์โหลดสำหรับ macOS ดูโค้ดบน GitHub

v1.1 · macos 13+ · apple notarized · free & open source · ~3mb

ใช้ยังไง

1

กดปุ่ม Fn ค้างไว้ แล้วพูดสิ่งที่อยากพิมพ์

2

ปล่อยปุ่ม — เสียงถูกถอดเป็นข้อความด้วย Whisper ภายในไม่กี่วินาที

3

AI แก้คำผิด เว้นวรรคให้ แล้วพิมพ์ลงแอปที่คุณใช้อยู่ทันที

ทำไมต้อง whisperapp

🎙️

Hold to Talk

กด Fn ค้างเพื่อพูด ปล่อยเพื่อหยุด — เหมือน walkie-talkie หรือสลับเป็นโหมด toggle (เคาะ 2 ครั้งเริ่ม เคาะครั้งเดียวหยุด) เปลี่ยนปุ่มลัดได้ตามใจ

⚡️

เร็วด้วย Groq

ถอดเสียงด้วย whisper-large-v3-turbo บน Groq — เร็วระดับเสี้ยววินาที มี free tier ใส่ API key เดียวใช้ได้ทั้งระบบ

✨

AI Correction

ข้อความที่ถอดได้ถูกขัดเกลาด้วย LLM อัตโนมัติ — แก้คำผิด เว้นวรรค จัดประโยคให้อ่านลื่น ทั้งไทยและอังกฤษ

📍

อยู่บน Menu Bar

ไม่มีหน้าต่างรก ไม่กินพื้นที่ Dock — แอปเงียบๆ อยู่บน menu bar พร้อมใช้ทุกเมื่อ ทำงานกับทุกแอปบน Mac

🔒

ปลอดภัย & Notarized

Sign ด้วย Developer ID และผ่าน Apple notarization — ดับเบิลคลิกติดตั้งได้เลย ไม่มีคำเตือนความปลอดภัย API key เก็บในเครื่องเท่านั้น

🪶

เบาหวิว

เขียนด้วย Swift ล้วน ไฟล์ติดตั้งเพียง ~3MB ไม่มี Electron ไม่มี background process หนักเครื่อง

เริ่มใช้ใน 2 นาที

  1. ดาวน์โหลด WhisperApp.dmg แล้วลากไปที่ Applications
  2. เปิดแอป → อนุญาตไมโครโฟน & Accessibility ตามที่ระบบถาม
  3. ขอ API key ฟรีที่ console.groq.com แล้ววางใน Settings (หรือตั้ง GROQ_API_KEY ใน ~/.zshrc)
  4. กด Fn ค้าง แล้วเริ่มพูดได้เลย
